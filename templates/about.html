Personal blog, description.

The blog was created from scratch on the Flask framework, written by one person with administrator rights. The posts highlight such a resource as Lorem Ipsum.

Site features:

- Only the administrator, the first created user in the database, has access rights to create and edit posts, delete comments;

- All registered users can leave comments under posts;

- Unregistered users can only view posts without the ability to comment.

Peculiar properties:

- To register and test the site, you can use any email, even non-existent (for example, user@mail.com);

- To obtain administrator rights and create or edit posts, use: (login admin@mail.ru, password 12345). Please do not delete existing posts;

- To upload an image to a post's cover page or body text form when creating a post, use the image address;

- The text from the resource Lorem Ipsum was used as filling the text in the posts. This is just a fictitious text of the printing and typesetting industry.

Used technologies:

- Templates Bootstrat

- Jinja

- HTML, CSS

- PostgreSQL

- WTForms

- Authentication

- GitHub / Version Control
